# Pages

A **page** is a Markdown file under `help/pages/`. Its logical ID is the path without the `.md` extension.

For example, this page's ID is `topics/pages`.

::include snippets/demo-tip.md

Developers author pages with optional YAML front matter. See [Authoring pages](help:authoring/pages) for the file layout and metadata conventions.
